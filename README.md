# Sistema-Digital-de-Controle-de-Hora-Extra-

Desenvolvimento de um sistema em C para registro, cálculo e gerenciamento de horas extras de colaboradores.
O projeto tem como objetivo automatizar o controle de jornada, otimizar o acompanhamento das horas trabalhadas, gerar relatórios precisos e aumentar a eficiência na gestão de tempo. Desenvolvido em parceria com a Base27, integra aprendizado prático, boas práticas de programação e inovação tecnológica.
